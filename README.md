# Ambient Feed (public)

Auto-generated JSON feed of Google TV Backdrop / Ambient Mode photos, grouped by
category. Updated automatically; do not edit by hand.

- `manifest.json` — category index (id, name, count, generated timestamp)
- `categories/*.json` — the assets per category (`url-1080`, `url-4K`, `title`, `added`)

**Base URLs** (once GitHub Pages / jsDelivr is enabled):

```
https://<user>.github.io/ambient-feed-public/manifest.json
https://cdn.jsdelivr.net/gh/<user>/ambient-feed-public@main/manifest.json
```

The image links point to Google-hosted media; this repo is an index, not a mirror.
